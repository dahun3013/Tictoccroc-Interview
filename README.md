# Tictoccroc-Interview
 서버개발자 1차 면접과제

# ERD 설계
요구사항
1.각 지역에 매장을 보유하고 있다.
2.매장마다 수업이 존재한다.
3.각 수업은 하루에 한번 진행한다.
4.예약 가능 일자는 현재일로 부터 14일 이내이며 당일과 과거로는 예약할 수 없다.
5.한 수업에 참여할 수 있는 아이 수 가 제한 되어있다.
6.동일인이 동일 매장, 동일 수업에 중복하여 예약을 할 수 없다.

객체추출
1.매장
2.수업
3.부모

관계추출
1.예약
2.수업개설
3.예약이력
