개념모델의 데이터는 DB상에서 직접 넣어줌

예약하기 -> 매장 선택 -> 수업 선택 -> 예약
내 예약내역 확인 -> 예약 선택 -> 예약 취소
예약자 현황 -> 매장 선택 -> 수업 선택 -> 예약자 현황
예약 이력 현황 ->매장 선택 -> 수업 선택 -> 예약 이력 현황

매장별, 수업별 예약 API		부모이름, 매장이름, 수업이름, 예약일, 인원 수
매장별, 수업별 예약 취소 API		부모이름, 예약일
매장별, 수업별 예약자 현황 API	수업이름
매장별, 수업별 예약 이력 현황 API	수업이름

컨트롤러 = API
서비스 = 데이터 모델링(데이터 가공)
모델 = 개념 모델링 (클래스 정의 및 겟셋)
